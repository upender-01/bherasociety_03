# 📖 Overview

The **Bhera Society Frontend** is developed using **React.js**, **Vite**, and **Tailwind CSS** to deliver a fast, responsive, and user-friendly experience.

This project serves as the public-facing website of the organization and is designed with a focus on performance, scalability, accessibility, and maintainability.

# ⚙ Getting Started

Clone the repository

```bash
git clone https://github.com/upender-01/bherasociety_03
```

Navigate to the project

```bash
cd bherasociety-frontend
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
